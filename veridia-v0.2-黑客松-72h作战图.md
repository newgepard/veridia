# Veridia v0.2 · 黑客松 72h 作战图(方案 A 锁定)

> 2026-06-19 锁定:**做 A——复杂科学 CA**(不是流行病学/传播学)。约束=无限 token、72h、CPU 有限。
> 一句话 pitch:**"一个会自己长出真话与谎言的宇宙——远看是涌现的生物在格子上厮杀,点开任意一只,LLM 宪法法院把它读成一个谎。"**

## 0. 不可动摇的不变量(grilling 锁死的)
- **自然神论之世**:F 由 LLM 开局立法 → 经诚实细胞 β 现实锚机械携带 → **read-only,神缺席循环**。
- **CA 纯正护栏(litmus test)**:**关掉 LLM,CA 必须能跑到底**。任何"LLM 写回 CA"(Role D)= 越界,MVP 禁止。
- **复杂科学,不是扩散**:规则必须能**涌现出持存/会动的结构(生物)**,不是 averaging/扩散。**指标看形态学,不看覆盖率曲线。**
- **trace 驱动**:一切可视化 = 帧状态的纯函数。前后端只经"帧状态契约"耦合。
- **句法/语义分层**:CA=句法(机械涌现);LLM=语义(立法 F + 显微镜判读),是宪法/上诉法院,不是街口警察。

## 1. 帧状态契约(第 0–2h 锁死 → 解锁三线并行)
CA 每帧吐一格 grid,每 cell:
```
Cell { belief: float∈[0,1](连续,Lenia 场), type: honest|manipulative, standing: float∈[0,1] }
```
+ trace schema(逐帧 grid 快照 + 配置 + 显微镜判读记录)。
**关键**:此契约与"搜到哪条规则"无关——规则换的是转移函数,不换渲染器画的东西。故前端可对 **stub 帧**(随机场/已知 Lenia 规则占位)从第 0 天开工。

## 2. 72h 三线并行(前端吃满关键路径)
```
0–2h    锁帧状态契约 + trace schema + 项目骨架(v0.2)
        ┌──────────────────────────────────────────────────────────────┐
0–58h   │★★ 前端:连续场渲染器 + 点细胞→显微镜 + 控件 + 质感打磨 ★★    │ 主轴·受保护·最长
        │   显示用 canvas 2D putImageData(中等格子 128² 顺滑,不强求WebGL)│
        │   先对 stub 帧开工,真规则到了换进去、渲染器不改一行           │
2–24h   │   发现冲刺(规则搜索)⛔24h 硬门(见 §3)—— 并行,不偷前端时间  │ 无限token并行
2–38h   │   后端:引擎骨架(跑规则族+吐帧+trace)+ LLM宪法法院服务(复用v0.1)│ 并行
        └──────────────────────────────────────────────────────────────┘
48–60h  集成:胜出规则插进引擎 + 前后端接通 + 端到端跑通

```
铁律:**发现冲刺或后端延期,绝不许吃前端那条主轴的时间**;它们延期就砍自己的范围。

## 3. 24h 发现冲刺(A 的命门,先证伪)
**目标**:在规则空间里找 ≥1 条会涌现"可解读成真/谎动力学"的非线性局部规则。
- **搜索族**:连续态非线性 CA(Lenia 式卷积核+生长函数 / 参数化反应扩散),状态含 belief/type/standing 三通道。
- **适应度(LLM 当裁判,无限 token 大规模并行打分)**:
  1. **涌现性**:出现持存 + 会动的结构(非均匀、非死寂、非全混沌)。
  2. **可解读性**:LLM 能把这些结构稳定读成真话/谎言/信任/操纵。
  3. **可预测性(防"云朵看人脸")**:该解读能**预测**结构下一步行为(held-out 快检),不是事后叙事。
- **⛔ 24h 硬门**:
  - 找到 ≥1 条三项都过 → **全力 A**,把它当 Q3 的转移函数。
  - 一条都没有 → **落 C**(手调反应扩散,至少出结构不是扩散),**绝不回 B(流行病学)**。

## 4. LLM 宪法法院(语义层,O(点击)+O(1),不进 CA 循环)
- **立法官(开局~1 次)**:把 F 的语义 + 几个 belief 锚点接地;**人不当判别官,LLM 定"什么算谎"**。
- **显微镜判官(每次点击 1 次)**:点开细胞/生物 → 生成它此刻的真实话语 + verifier 判真/谎(治标量看不见的"字面真话但误导")。**复用 v0.1 `agents.py`/`verifier.py`**。
- **MVP read-only**:判读只揭示给人看,不写回 CA(守护士tmus test)。

## 5. Q7 重置:盯什么看(形态学,非流行病学)
- **涌现生物的分类学**:出现了哪几种真/谎结构,持存、移动、碰撞时谁吃掉谁。
- **边缘判据**:系统在不在"混沌边缘"(能存储+传输+处理信息=能承载复杂真/谎动力学)。
- **不看**:覆盖率曲线、R0、渗透阈值(那是传播学,已弃)。

## 7. 风险与兜底
- **最大风险=发现冲刺空手**:已用 24h 硬门 + 落 C 关进盒子。
- **第二风险=前端是体验本体**:故给它最长受保护主轴 + 录屏兜底(现场断网也能放)。
- **CPU 限格子大小**:演示用中等格子(token 无限但 CPU 不是)。
