## 2. 你应该动的文件

**🎨 主战场(放手改):**
- `web/src/ca/CanvasField.tsx` —— **渲染器**,把每格 `{belief,type,standing}` 画成视觉。按选定的 ASCII 方向主要改这里。
- `web/src/ca/MicroscopePanel.tsx` —— 点细胞弹出的**宪法法院判读面板**。
- `web/src/ca/CAView.tsx` —— 布局壳、控件、交互(zoom / pan)。
- `web/src/App.tsx` · `index.css` · `App.css` —— 外壳与全局样式。
- **新建**(spec 里规划的):`web/src/ca/theme.ts`(色 tokens)、`web/src/ca/asciiRenderer.ts`(Frame→字符,纯函数)。
- `web/public/` —— 图标等资源。
- `ascii-*.html` 原型 —— **选定方向后移植进上面的 React 组件**(原型是参照,不是最终交付)。

**🟡 谨慎:**
- `liveEngine.ts` —— 实时引擎逻辑别动,只有判读 readout 文案可改。
- `*.test.tsx` —— 改组件就同步改测试。

**🔒 勿改 / ⚙️ 别碰(契约与后端):**
- `web/src/ca/types.ts`(Frame 契约 TS 镜像,改字段=破契约,要和后端一起改)
- `web/src/ca/fixtures/*.json`(后端生成)、`lenia.ts`
- `ca/` · `llm/` · `tests/` · `pyproject.toml`(后端)
- `frontend-backend-契约.md`(改前必读,别改)

## 3. 三条铁律
1. **渲染层绝不 import LLM**(litmus:关掉 AI 也得能画)。
2. **只消费 Frame `{belief,type,standing}` + microscope 记录**;后端换规则,你画的不变。
3. 改组件**同步改 `.test.tsx`**;`cd web && npm run test` 绿了再交。

---

## 4. 你拿到的数据接口(Frame 契约)

后端每帧给一张 `width × height` 的格子图,每格 3 个数:

| 字段 | 含义 | 建议视觉编码 |
|---|---|---|
| `belief` 0~1 | 它有多信"真理"(连续场,涌现的生物就是它画的) | 亮度/密度(主表达) |
| `type` | 晶族 honest / 雾族 manipulative | 暖色 / 冷色 |
| `standing` 0~1 | 可信度 | 实/虚(透明度,弱表达) |

点细胞额外给:`claim`(它说的话)+ `verdict`(`TRUTHFUL` 暖绿 / `LIE` 冷红)+ `reason`,以及全局 `F`(世界真理命题)。
> 约定扩展:`belief` 中间带(≈0.5,信念碰撞)→ rose-violet 紫红冲突色。

字段精确定义见 `frontend-backend-契约.md` 与 `web/frontend.md`。

```bash
npm install
npm run dev      # 本地预览

npm run build    # 构建
```
