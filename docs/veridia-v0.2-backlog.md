## ✅ `belief`(信念 / 正在传播的 claim)—— 已晋升 MVP 核心(不再是 backlog 项)
- 2026-06-19:B(信息传染)锁定后,`belief` 是模拟的实体本身,已进核心字段集 `{type, belief, standing}`。
- 取值:**连续标量 ∈ [0,1]**(0=信谎极、1=信真极),平滑信念波 = Lenia 连续美感。
- 全局锚 `F`=真相命题;约束层收益奖励 belief 贴近 F。

## 🥈 `reputation` / `score` 拆分(财富 vs 信任)
- **解锁**:把 MVP 合并的 `standing` 拆开 → "又富又越来越不被信"的高明骗子,欺骗戏剧的心脏。
- **成本**:1 个标量 + 规则微调。**最便宜的"加深主题"。**
- **触发**:跑出来发现骗子又富又横、demo 单薄 → 立刻拆。

## 🥉 `commitment`(可验证承诺)
- **解锁**:诚实细胞立公开可验证承诺,毁约=铁证谎;操纵者承诺没人信 → **透明从弱点变武器**,诚实能主动赢。
- **成本**:中(pledge 字段 + 校验)。
- **触发**:想回答"诚实凭什么赢",或想要"诚实领地反攻"的戏剧。
